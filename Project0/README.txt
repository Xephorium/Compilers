Execution Scripts:
    - To clean project, execute "clean.bat" on Windows or "clean.bash" on Linux.
    - To run project, execute "run.bat" on Windows or "run.bash" on Linux.
        > Run script automatically cleans project.
        > Run script handles parameters as if passed directly to P0 executable.

            Examples:
                run test\input
                run text\input.sp2020
                run < test\input.sp2020


Other Notes:
    - Project build (.class) files are created in the "build" directory.
    - Project output (.xorder) files are created in the "out" directory.
